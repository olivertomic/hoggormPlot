Convenience plotting functions
==============================

This is an overview of available **convenience plotting functions** in 
HoggormPlot. They are based on the main plot function 
``hoggorm.main_plot.plot`` and provide simplified interfaces for interactive
use.

-----------------------

.. note:: The two functions below, i.e. ``.explVar`` and ``.explainedVariance``
          are two options of how to generate explained variance plots. They 
          generate identical plots when provided the same input paramters
          or arguments.
